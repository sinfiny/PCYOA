# Interactive CYOA Creator Plus

Interactive CYOA Creator Plus is a Svelte-based editor and viewer for interactive CYOA projects.

## Status

Stable version: [ICC Plus 2](https://hikawasisters.neocities.org/ICCPlus2/)

Legacy version: [ICC Plus](https://hikawasisters.neocities.org/ICCPlus/) is no longer maintained.

## Development

The app is in `ICCPlus`.

```sh
cd ICCPlus
npm install
npm run dev
```

Useful commands:

```sh
npm run build
npm run preview
npm run check
```
